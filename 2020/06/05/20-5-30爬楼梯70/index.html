20-5-30爬楼梯70

发表于 2020-06-05 阅读次数：

题目

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？注意：给定 n 是一个正整数。

示例 1：

输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。

示例 2：

输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。

思路 1 —错误！

把相邻的1合并成2，有多少种合并方法，就有多少种走法

和全部计算的方法本质上是相同的，每次有一步、两步、三步……走了两个台阶，最后是有n/2步走了两个台阶

奇偶数的边界不同，偶数如6最后走3步2个台阶，7走3步2个台阶和1步一个台阶，这就有A44/A33个走法

假设有n个台阶，从0开始循环，循环到n/2，6 到 3，7到3 ，

0个2，6-0=6个1，7-0=7个1

1个2，6-2=4个1，7-1=5个1

.

3个2，6-6=0个1，7-2*3=1个1

比如是 6个台阶

程序如下：

注意这里容易出现int越界，最好用long类型的

官方解法1：暴力递归

public int climbStairs(int n) {
    climb_Stairs(0, n);
}
public int climb_Stairs(int i, int n) {
    if (i > n) {
        return 0;
    }
    if (i == n) {
        return 1;
    }
    return climb_Stairs(i + 1, n) + climb_Stairs(i + 2, n);
}

官方解法2– 动态规划

第 n阶可以由以下两种方法得到：

  • 在第 (n-1) 阶后向上爬一阶。

  • 在第 (n-2) 阶后向上爬 2 阶。

所以到达第 ii 阶的方法总数就是到第 (n-1) 阶和第 (n-2)阶的方法数之和。

精选解法1：动态规划 更简洁

想知道n个台阶有多少种走法，只需要第n-1个台阶和第n-2个台阶有多少种走法
